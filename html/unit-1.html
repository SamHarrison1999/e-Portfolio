Benefits of OOP

  • Shorter methods
  • Supports public and private properties
  • Loose coupling
  • Supports code development distribution between developers
  • Improved maintainability
  • More sustainable
  • Reusability

Four pillars of object-oriented programming

  • Abstraction
    • Abstraction allows you to hide an objects' complexity showing only the necessary features of an object.
  • Encapsulation
    • Encapsulation is the process of dividing the code into a public interface and a private implementation of that interface.
  • Inheritance
    • Inheritance allows you to reuse code by extending a class. The child classes contain all the attributes and methods of their parent class.
  • Polymorphism
    • An example of polymorphism is overloading methods so they work with different data types

Defining a class in Python

The syntax for declaring a class in python is the class keyword followed by the class name followed by a colon. A class can be thought of as a blueprint for an object. The code below shows a simple person class.

class Person:

    Represents a person
    attributes: forename, surname
    """

Access modifiers in python classes

  • Public
    • Anything that is public can be accessed from anywhere in your application.
    • By default, in Python, all attributes and methods of a class are public.
  • Private
    • Anything that is private is only accessible within the class.
    • Python's convention for naming private variables and methods is a single underscore at the start of the method or variable.
    • This is only a convention as Python doesn't really recognise private variables and methods.

The code block below is an example of a public class with public attributes and a public method.

class Person:
    """
    Represents a person
    attributes: forename, surname
    """
    def __init__(self, forename: str, surname: str) -> None:
        # Public attributes
        self.forename = forename
        self.surname = surname

    # Public method
    def name(self) -> str:
        """Return a persons first name and last name"""
        return self.forename + " " + self.surname

>>> person = Person("Homer", "Simpson")
>>> print(person.forename + " " + person.surname)
Homer Simpson
>>> print(person.name())
Homer Simpson

The code block below is an example of a private class

class Person:
    """
    Represents a person
    attributes: forename, surname
    """
    def __init__(self, forename, surname):
        # Private attributes
        self._forename = forename
        self._surname = surname

    def getForename(self):
        """Return a persons first name"""
        return self._forename

    def getSurname(self):
        """Return a persons last name"""
        return self._surname


>>> person = Person("Homer", "Simpson")
>>> print(person.getForename() + " " + person.getSurname())
Homer Simpson

15.2 Attributes

Setting the value of an attribute

You can assign values to attributes using the dot notation.

>>> blank.x = 3.0
>>> blank.y = 4.0

Reading the value of an attribute

You can read the value of an attribute for an instance using the dot notation.

>>> blank.y
4.0
>>> x = blank.x
>>> x
3.0

blank.x returns the value of the x attribute for the blank object.
blank.y returns the value of the y attribute for the blank object.

Using the dot notation as part of an expression

>>> '(%g, %g)' % (blank.x, blank.y)
'(3.0, 4.0)'
>>> distance = math.sqrt(blank.x**2 + blank.y**2)
>>> distance
5.0

Passing an instance as an argument

def print_point(p: point) -> None:
    """print the x and y coordinates for a point"""
    print('(%g, %g)' % (p.x, p.y))

>>> print_point(blank)
(3.0, 4.0)

In the code above p is an alias representing a point object.

15.4 Instances as return values

A function can also return an instance of an object as can be seen from the find center function below.

def find_center(rect: Rectangle) -> Point:
    """Find the x and y coordinates for the center of a rectangle"""
    p = Point()
    p.x = rect.corner.x + rect.width/2
    p.y = rect.corner.y + rect.height/2
    return p

>>> center = find_center(box)

15.6 Copying

Copying an object ensures changes to an object in one place don't have unintended consequences in another place. A copy of an object is not the same object. You need to perform a deep copy to copy embedded objects.

>>> p1 = Point()
>>> p1.x = 3.0
>>> p1.y = 4.0
>>> import copy
>>> p2 = copy.copy(p1)
>>> print_point(p1)
(3, 4)
>>> print_point(p2)
(3, 4)
>>> p1 is p2
False
>>> p1 == p2
False
>>> box2 = copy.copy(box)
>>> box2 is box
False
>>> box2.corner is box.corner
True
>>> box3 = copy.deepcopy(box) # box and box3 are completely separate objects
>>> box3 is box
False
>>> box3.corner is box.corner
False

Exercise 15.1

  • Create a circle class with the attribute's center and radius, where center is a point object and radius is a number.
  • Create a Circle object with its center at (150,100) and radius 75.
  • Write a function named point_in_circle that takes a circle and a point and returns true if the point lies in the circle or is on the boundary of the circle.
  • Write a function named rect_in_circle that takes a circle and a rectangle and returns true if the rectangle lies entirely in the circle or on the boundary of the circle.
  • Write a function named rect_circle_overlap that takes a Circle and a Rectangle and returns True if any of the corners of the Rectangle fall inside the circle. Or as a more challenging version, return True if any part of the Rectangle falls inside the circle.
import math
import copy

class Point:
    """Represents a point in 2-D space."""

    def __init__(self, x = 0, y = 0) -> None:
        """Constructor for a point object"""
        self.x = x
        self.y = y

    def __str__(self) -> str:
        """Return the x and y coordinates for a point"""
        return '(%g, %g)' % (self.x, self.y)

class Rectangle:
    """
    Represents a rectangle.
    attributes: width: int, height: int, corner: Point.
    """
    height = None
    width = None
    corner = None

class Circle:
    """
    Represents a circle.
    attributes: center: Point, radius: float
    """
    center = None
    radius = None

def distance_between_points(p1: Point, p2: Point) -> int:
    """Calculate the distance between 2 points"""
    dx = p1.x - p2.x
    dy = p1.y - p2.y
    dist = math.sqrt(dx ** 2 + dy ** 2)
    return dist

def point_in_circle(point: Point, circle: Circle) -> bool:
    """Check if a point is in the circle"""
    d = distance_between_points(point, circle.center)
    return d <= circle.radius

def rect_in_circle(rect: Rectangle, circle: Circle) -> bool:
    """Check if the rectangle is in the circle"""
    p = copy.copy(rect.corner)
    if not point_in_circle(p, circle):
        return False
    p.x += rect.width
    if not point_in_circle(p, circle):
        return False
    p.y -= rect.height
    if not point_in_circle(p, circle):
        return False
    p.x -= rect.width
    if not point_in_circle(p, circle):
        return False
    return True

def rect_circle_overlap(rect: Rectangle, circle: Circle) -> bool:
    """Check if the rectangle overlaps the circle"""
    p = copy.copy(rect.corner)
    if point_in_circle(p, circle):
        return True
    p.x += rect.width
    if point_in_circle(p, circle):
        return True
    p.y -= rect.height
    if point_in_circle(p, circle):
        return True
    p.x -= rect.width
    if point_in_circle(p, circle):
        return True
    return False

>>> box = Rectangle()
>>> box.width = 100.0
>>> box.height = 200.0
>>> box.corner = Point()
>>> box.corner.x = 50.0
>>> box.corner.y = 50.0
>>> circle = Circle()
>>> circle.center = Point()
>>> circle.center.x = 150.0
>>> circle.center.y = 100.0
>>>circle.radius = 75.0
>>> print(point_in_circle(box.corner, circle))
False
>>> print(rect_in_circle(box, circle))
False
>>> print(rect_circle_overlap(box, circle))
True

def draw_circle(t, circle):
    """Draw the circle"""
    t.color("black") # Set colour
    t.hideturtle() # Hide turtle
    t.pu() # Lift pen
    t.goto(circle.center.x, circle.center.x - circle.radius)  # Position pen
    t.pd()  # Place pen to draw
    t.circle(circle.radius)  # Draw the circle
    t.pu()  # Lift pen

def draw_rect(t, rect):
    """Draw the rectangle"""
    t.color("black") # Set colour
    t.hideturtle() # Hide turtle
    t.pu() # Lift pen
    t.goto(rect.corner.x, rect.corner.y)
    t.setheading(0)
    t.pd() # Place pen
    # Draw Rectangle
    for length in rect.width, rect.height, rect.width, rect.height:
        t.fd(length)
        t.rt(90)
