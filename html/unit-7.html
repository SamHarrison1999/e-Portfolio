Discuss the ways in which data structures support object-oriented development. Use examples of three different data structures to contextualise your response.

In object-oriented programming, you use classes to represent real world objects. Each class has a set of methods and attributes attached to it. Instances of objects can be stored in data structures, for example, a list of people. Data structures can also be attributes of a class, for example, a person class may use a dictionary to store their height and weight. A dictionary is used to store key value pairs so they can be indexed in a persons class, for example, measurements would be the dictionary then inside the dictionary there would be a key for height, waist size etc. A tuple can be used in a similar way to a list with the main difference being that it is immutable.

def print_nested_dictionary_keys_and_values(dictionary):
    for key, value in dictionary.items():
        if isinstance(value, dict):
            print_nested_dictionary_keys_and_values(value)
        else:
            print(f'{key} : {value}')


def nested_dictionary_values(dictionary):
    for values in dictionary.values():
        if isinstance(values, dict):
            yield from nested_dictionary_values(values)
        else:
            yield values
